Paste Individual Response Codes

Ask each PLC member to complete the same check-in and share their response code if they choose to participate. Paste one code per line.

Facilitator note: The dashboard should be used to guide conversation, not to prove whether the team is “good” or “bad” at PLC work. Begin with evidence, notice where perceptions differ, and then choose one realistic next step.

One check-in at a time: all pasted codes must come from the same check-in (for example, all Shared Purpose codes, or all Full Compass codes). Mixed check-in codes cannot be combined into one report.
