The Hub: Your Front Door to the Tool Suite

The home page is the hub for the PLC Readiness Compass. Rather than one long survey, the Compass is a small suite of tools you can mix and match: a planner, a check-in menu, a team dashboard, and this guide. The menu version was created to reduce fatigue and support timely, relevant reflection, so a team can choose one short, targeted check-in when that is all the moment calls for.

When to use the Planner

When you are unsure which check-in fits the current moment. The planner suggests a starting point and approximate windows across the year. Treat it as a suggestion, not a required schedule.

When to use a short Check-In

During the year, when the team needs a focused, low-fatigue conversation about one readiness condition. Usually 3–5 minutes.

When to use the full Compass

For a beginning-of-year baseline, a midyear reset, a new or restarting PLC, or a major team transition, when the whole picture matters. Roughly 10–15 minutes.

When to use the Dashboard

After members complete the same check-in, to turn voluntarily shared response codes into a conversation report that surfaces patterns and perception gaps.

Check-ins can be used independently and do not need to be completed in order. Pick the one that fits where the team is right now.

The original full Compass tool remains available through a single text link on the home page. It is preserved so earlier work is not lost, but it is no longer the main navigation pathway; the hub points facilitators toward the pathway that fits their current need.

Tool Overview

The PLC Readiness Check-In Menu is a structured reflection and conversation tool. It helps a team notice the cultural, structural, and leadership conditions that support or constrain authentic professional learning community work.

The tool assumes that PLCs may be formally facilitated by a department lead, instructional coach, or administrator, while still requiring shared ownership, teacher voice, and collective responsibility for student learning.

The tool should not be used to evaluate individual teachers. Its purpose is to surface patterns, support honest conversation, and guide one manageable next step.

Choosing a Check-In

The PLC Year Planner can suggest a check-in based on your current context and, if you add a start date, sketch approximate windows across the year. Use the planner to select a check-in, but do not treat the planner as a required schedule. It is a starting suggestion, not a calendar to comply with.

What Makes the Ratings Meaningful?

The ratings are developmental and conversation-based, not psychometric. Because the tool is designed for practitioner use, ratings should be interpreted as evidence-informed perceptions rather than validated measurement scores.

Useful evidence might include agendas, student work conversations, shared resources, instructional decisions, follow-up actions, team norms, access to data, or moments when trust either supported or limited collaborative inquiry.

Exporting Reports as PDFs

The individual check-in and the team dashboard both include an export button. The tool uses the browser’s built-in print-to-PDF function so reports can be saved without storing responses on a server.

After selecting export, choose Save as PDF in the print destination menu.

Recommended Process

Step Facilitator Move Approx. Time
1. Choose the check-in Decide whether the team needs the Full Compass or one short check-in. Share the core question so everyone knows the focus. 2–3 min
2. Frame the purpose Explain that the tool is developmental, not evaluative. Emphasize voluntary sharing and collective learning. 3–5 min
3. Individual check-in Members complete the check-in privately and think of evidence before selecting ratings. 3–15 min
4. Generate team dashboard Paste voluntarily shared codes from the same check-in into the dashboard and generate the report. 3–5 min
5. Discuss evidence Start with evidence before solutions. Ask, “What examples might explain this pattern?” 10–15 min
6. Select one next step Complete the action plan. Choose one readiness condition to strengthen before the next PLC cycle. 5–10 min

Interpretation Guidance

Average Score Readiness Level Recommended Response
1.0–1.9 Foundational Concern Begin with structures, trust, norms, and clarity before expecting deeper PLC work.
2.0–2.79 Emerging Identify one condition to strengthen and create a short-term action step.
2.8–3.49 Developing Use existing strengths to deepen inquiry, application, and follow-up.
3.5–4.0 Established Move toward more advanced PLC practices such as shared inquiry cycles, peer feedback, or instructional artifacts.

Facilitation Norms

  • Discuss team conditions, not individual performance.
  • Ask for evidence before interpreting scores.
  • Treat disagreement as useful data.
  • Choose one next step rather than trying to fix every domain at once.
  • Revisit the action plan at a future PLC meeting.

Suggested Rationale Language

The tool is intentionally serverless. It does not store or transmit individual responses. Members choose whether to share their response code, and the team dashboard emphasizes collective patterns rather than individual evaluation. This design choice supports trust and psychological safety, both of which are important readiness conditions for PLC work.

The menu version adds shorter, single-domain check-ins so teams can reflect on one condition at a time. This reduces survey fatigue and keeps reflection timely and relevant, while the full Compass remains available as a comprehensive baseline.

The PDF export feature uses browser-based print-to-PDF rather than server-side storage. This allows teams to preserve a report for meeting use while maintaining the tool’s privacy-conscious design.

Framework Reference

This tool is grounded in Hord and Sommers’ five components of professional learning communities: shared and supportive leadership, shared values and vision, collective learning and application, shared personal practice, and supportive conditions (Hord & Sommers, 2008).

Note on language: in the menu, the component Hord and Sommers call Shared Values and Vision appears as Shared Purpose for clarity with practitioners. The framework name is retained here so the connection to the original five components remains clear.

Hord, S. M., & Sommers, W. A. (2008). Leading professional learning communities: Voices from research and practice. Corwin Press.
